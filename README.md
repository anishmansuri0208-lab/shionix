# 🛍️ Shionix — Customer Website

**Smart Products, Smart Shopping**

Live e-commerce website for Shionix brand.

## 🚀 GitHub Pages pe Deploy Kaise Karein

### Step 1 — Repository Banao
1. [github.com](https://github.com) pe login karo
2. Top-right mein **"+"** icon → **New Repository**
3. Repository name: `shionix-website`
4. **Public** select karo
5. **Create Repository** click karo

### Step 2 — Files Upload Karo
1. **"uploading an existing file"** link click karo
2. Is folder ki **saari files** drag & drop karo
3. Neeche **"Commit changes"** click karo

### Step 3 — GitHub Pages Enable Karo
1. Repository mein **Settings** tab click karo
2. Left sidebar mein **Pages** click karo
3. **Source** mein → **Deploy from a branch** select karo
4. **Branch** → `main` select karo → **Save**
5. 2-3 minute wait karo
